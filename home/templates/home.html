How it works

First the user can make a site by entering the name of the site and hittng the go button or simply typing the name on the url.
User flow
  1. If the site is already taken the user is shown a box to enter the password to decrypt the site.
  2. If the password is correct the user can access the site and update and delete text or pages.
  3. If the password is incorrect there is no way of seeing the data other than brute-forcing the password.
  4. If the site is not taken a fresh site is shown where the user can enter data and save it using a password

    5. The data can be accessed and modified anywhere using the same password
